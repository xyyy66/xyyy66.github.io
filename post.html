代码高亮演示 (Code Highlighting Demo)

下面是一个使用 Python 编写的简单示例，我们使用 Prism.js 进行多语言的语法高亮：

# Python Code Example
def fibonacci(n):
    """
    Generate Fibonacci sequence up to n
    """
    sequence = [0, 1]
    while len(sequence) < n:
        sequence.append(sequence[-1] + sequence[-2])
    return sequence

print("Fibonacci sequence of 10:", fibonacci(10))
